# SQL-Data-Analysis


## 📌 Project Overview

This project demonstrates SQL data analysis using MySQL Workbench on an E-commerce Orders dataset. It covers database creation, data import, SQL querying, and business insight generation using SQL.

---

## 🎯 Objectives

- Create a MySQL database
- Import CSV data into MySQL
- Perform SQL queries for data analysis
- Generate business insights from the dataset

---

## 🛠️ Tools Used

- MySQL Workbench
- SQL
- Git & GitHub

---

## 📂 Dataset Information

- **Database Name:** `ecommerce_db`
- **Table Name:** `orders`
- **Dataset:** E-commerce Orders Dataset
- **Total Records:** 1200

---
